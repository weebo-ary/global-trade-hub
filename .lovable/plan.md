

# HKR Global Trade - Premium Export Portfolio Website

## Project Summary
A Fortune 500-style corporate website for HKR Global Trade featuring 8 fully-designed pages, premium glassmorphism design, demo lock functionality, and comprehensive B2B export features. All forms will function as UI demos with success messages, and realistic placeholder content will be used throughout.

---

## Design Foundation

### Visual Theme
- **Primary Navy** (#1e3a5f) for trust and corporate feel
- **Accent Green** (#22c55e) for growth and freshness
- **Gold accents** (#d4af37) for certification badges
- Clean white backgrounds with glassmorphism cards
- Smooth fade-in animations and parallax scrolling effects

### Global Components
- **Demo Banner**: Auto-scrolling marquee at top of all pages ("⚠️ This is a Demo Website – For Presentation Purposes Only")
- **Sticky Header**: Logo, navigation, language selector (UI-ready), and prominent green "Get Quote" button with blur effect on scroll
- **Demo Lock System**: Configurable toggle (`unlockAllSections = false`) that shows first 2 sections on non-home pages, with professional unlock overlay
- **Footer**: Full contact info, certification badges, social links, download buttons, and quick navigation

## Interactive Features

### Forms (Demo Mode - UI Only)
- **RFQ Form**: Company name, country, product category, specific products, quantity, target price, delivery timeline, contact details
- **Contact Form**: Name, email, company, message
- **Distributor Application Form**: Company details, experience, regions of interest
- All forms show professional success messages with "We'll respond within 24 hours"

### Floating Elements
- WhatsApp chat button (bottom right)
- Scroll-to-top button (appears on scroll)
- Floating "Download Brochure" sidebar button

### PDF Downloads
- **Company Profile PDF**: Generated document with company overview, certifications, and contact info
- **Product Catalog PDF**: Generated document with product listings by category

### Premium UX Features
- Language selector dropdown (EN, AR, ES, DE, FR options - UI ready)
- Currency/units toggle (USD/EUR/INR - UI display)
- Product filtering by category
- Smooth scroll navigation
- Mobile hamburger menu with slide-out drawer
- Loading animations and number counters

## Technical Implementation
- React with React Router for 8-page navigation
- Tailwind CSS with custom navy/green/gold theme
- Glassmorphism card components
- CSS animations and parallax effects
- React Hook Form with Zod validation for forms
- Lucide React icons throughout
- PDF generation for downloadable resources
- Fully responsive mobile-first design
- Demo lock configuration system
